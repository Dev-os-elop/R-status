# Changelog

## 0.5.5 - 2026-07-10

- 기본 메뉴바 테마가 Cat Original인지 재확인
- Appearance 선택과 관계없이 앱·알림 아이콘을 흰 고양이 Cat Original로 고정
- Cat Silhouette 등 다른 테마는 메뉴바 상태 아이콘에만 적용

## 0.5.4 - 2026-07-10

- 상태 글자와 함께 표시되는 메뉴바 아이콘을 19px에서 24px로 확대
- 고양이 테마의 아이콘 내부 여백을 8%에서 3%로 축소
- Cat Original 얼굴을 넓고 낮은 비율로 조정해 세로로 길어 보이던 문제 수정
- 대기 상태와 실행·완료·중단·실패 상태 사이의 아이콘 크기 차이 축소

## 0.5.3 - 2026-07-10

- Cat Original의 메뉴바 고양이 얼굴을 약 8% 확대
- Cat Original과 Cat Silhouette의 앱 아이콘 얼굴 여백 축소
- 상태 보조 기호와 겹치지 않도록 메뉴바 얼굴·기호 간격 유지

## 0.5.2 - 2026-07-10

- 사용자가 선택한 흰 고양이 디자인을 Cat Original 테마로 적용
- Cat Original에 분홍 귀·코, 짧은 수염, 상태별 표정과 외곽선 색상 적용
- 대기 점, 실행 회전 화살표, 완료 체크, 중단 일시정지, 실패 느낌표 보조 기호 적용
- 새 설치의 기본 테마와 앱 번들 아이콘을 Cat Original로 변경
- 초단순 눈 상태 디자인은 Cat Silhouette 선택 옵션으로 유지

## 0.5.1 - 2026-07-10

- Appearance에 Cat Outline과 Cat Silhouette 아이콘 테마 추가
- 두 고양이 테마 모두 입·코·수염 없이 머리 윤곽, 이마 무늬, 눈만 표시
- 대기·실행·완료·중단·실패를 눈의 색과 원·재생·체크·일시정지·X 모양으로 구분
- 새 설치의 기본 테마를 Cat Silhouette으로 변경
- 앱 번들 아이콘을 파란 배경의 독자적인 고양이 실루엣으로 변경

## 0.5.0 - 2026-07-10

- 별도 창 없이 기존 메뉴 오른쪽에 펼쳐지는 Settings 서브메뉴 추가
- Basic에서 시스템 언어·한국어·영어 선택 지원
- Appearance에서 Status Pulse, Progress Blocks, Signal Orbit, Window Check, Layered S 선택 지원
- Advanced에서 메뉴바 실행 시간과 로그인 시 실행 토글 지원
- 선택한 아이콘을 메뉴바 상태별 색상과 앱 아이콘에 즉시 적용
- RStudio 상표 아이콘 제거 및 독자적인 Status Pulse 앱 아이콘 적용
