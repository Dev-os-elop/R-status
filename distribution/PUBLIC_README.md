# ES Status Downloads

ES Status는 RStudio 코드 실행 상태를 macOS 메뉴바와 알림으로 보여줍니다.

실행 중인 R 작업의 전체 CPU 사용률, R process, parallel worker 수도 메뉴에서 확인할 수 있습니다.

## 설치

설치와 소스는 [Dev-os-elop/R-status](https://github.com/Dev-os-elop/R-status)에서 제공합니다.

1. 저장소에서 **Code → Download ZIP**을 선택합니다.
2. 압축을 풀고 **Install ES Status.command**를 실행합니다.
3. 설치 완료 후 RStudio를 재시작합니다.
4. RStudio에서 **Addins → Run Selection with Status**를 실행합니다.

필수 조건: macOS 13 이상, R, RStudio Desktop.

설치기는 GitHub Release의 사전 빌드 Apple Silicon 실행 파일을 내려받아 설치하므로 Xcode가 필요하지 않습니다.

RStudio는 Posit Software, PBC의 상표입니다. 이 프로젝트는 Posit의 공식 제품이 아니며 RStudio 로고를 포함하거나 사용하지 않습니다.
