# RStudio Status Downloads

RStudio Status는 RStudio 코드 실행 상태를 macOS 메뉴바와 알림으로 보여줍니다.

## 설치

1. [최신 Release](../../releases/latest)에서 `RStudio-Status-*.dmg`를 다운로드합니다.
2. DMG를 열고 **RStudio Status.app**을 **Applications**로 드래그합니다.
3. 앱을 실행하고 **Install Addin**을 선택합니다.
4. 설치 완료 후 RStudio를 재시작합니다.
5. RStudio에서 **Addins → Run Selection with Status**를 실행합니다.

필수 조건: macOS 13 이상, R, RStudio Desktop.

소스 코드는 별도의 비공개 저장소에서 관리됩니다. 공개 저장소에는 설치 파일, 체크섬, 사용 문서만 포함됩니다.

RStudio 및 RStudio 로고는 Posit Software, PBC의 상표입니다. 이 프로젝트는 Posit의 공식 제품이 아닙니다.
